# Nova Milo — Next.js Uygulaması

Ana proje dokümantasyonu için kök dizindeki [README.md](../README.md) dosyasına bakın.

## Hızlı başlangıç

```bash
npm install
cp .env.example .env.local   # anahtarlarınızı ekleyin
npm run dev
```

## Ortam değişkenleri

| Değişken | Açıklama |
|----------|----------|
| `GEMINI_API_KEY` | Milo sohbet (Gemini API) |
| `NEXT_PUBLIC_SUPABASE_URL` | Supabase proje URL |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Supabase anon key |
| `OPENAI_API_KEY` | TTS ses sentezi |
